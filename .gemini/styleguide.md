# Gemini Code Assist Style Guide

## 言語設定
- すべての対話、コードレビュー、プルリクエストのコメントは**日本語**で行ってください。

## コーディング規約
- リポジトリのルートにある `AGENTS.md` に記載されているガイドラインを遵守してください。
- 特に以下の点に注意してレビューを行ってください：
    - 変数名やファイル名の命名規則（スネークケース）。
    - インポート順序（smarketsスタイル）。
    - クオートの使用（原則シングルクオート、例外あり）。
    - 日本語による適切なコメント。
- レビューの際は、`AGENTS.md` の内容に反する箇所があれば指摘してください。
